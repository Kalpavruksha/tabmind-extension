## Step-by-Step Installation

### 1. Open Chrome Extensions Page
- Open Google Chrome
- Type `chrome://extensions/` in the address bar
- Press Enter

### 2. Enable Developer Mode
- Look for the **"Developer mode"** toggle in the top-right corner
- Click it to turn it **ON**
- You should see additional buttons appear

### 3. Load the Extension
- Click the **"Load unpacked"** button
- Navigate to the `tabmind` folder (this folder)
- Click **"Select Folder"**

### 4. Verify Installation
- You should see the TabMind extension card appear
- The TabMind icon (ðŸ§ ) should appear in your Chrome toolbar
- If you don't see it, click the puzzle piece icon and pin TabMind

### 5. Start Using TabMind!
- Click the TabMind icon to see the popup
- Press `Ctrl+Shift+T` (Mac: `Cmd+Shift+T`) to open the side panel
- Let TabMind analyze your tabs and start getting insights!

## Troubleshooting

### Extension doesn't appear
- Make sure Developer Mode is enabled
- Try reloading the extension (click the refresh icon on the extension card)
- Check the Chrome console for errors

### Side panel doesn't open
- Make sure you're using Chrome version 114 or later
- Try clicking the TabMind icon and then "Open Dashboard"
- Restart Chrome if needed

### Icons not showing
- The icons should be in the `icons/` folder
- If missing, check that all files were copied correctly
